## 💾 记忆同步检查 ✅

**每次心跳检查：**

- [ ] 检查 Git 同步状态

- [ ] **读取所有关键信息** ⭐ 新增（队列模式）

### 🎯 同步命令（从简单到完整）

**命令1：仅同步记忆**
```bash
bash /home/micker/clawd/sync-all-sisters.sh
```
**快速触发：** 聊天中说"同步所有"或"保存大家"
**特点：** 快速同步，不读取关键信息

**命令2：完整同步（推荐）** ⭐⭐⭐
```bash
bash /home/micker/clawd/sync-complete.sh
```
**功能：**
- ✅ 同步所有姐妹的记忆
- ✅ 读取**所有关键信息**（全面版 v2.0）
- ✅ 更新关键信息汇总
- ✅ 提交到 GitHub
- ✅ **队列模式，不打断当前任务** 🆕

**快速触发：** 聊天中说"完整同步"、"全部同步"或"彻底同步"

**特点（v2.0 新增）：**
- 🆕 **队列机制**：不会打断本小姐当前正在思考的内容
- 🆕 **开发进度**：自动读取记忆系统开发进度
- 🆕 **开发文档**：自动汇总所有开发文档
- 🆕 **日志位置**：自动记录所有日志文件路径

**自动读取的关键信息（v2.0 全面版）：**
- 🔑 API Keys（姐妹们的发帖密钥）
- 🤖 Agent IDs（姐妹们的 Agent ID）
- 🔗 Session Keys（@召唤系统的会话密钥）
- 🌐 Gateway 配置（各姐妹的独立 Gateway 端口和日志）
- 🖥️ 服务器配置（IP、SSH端口等）
- 📁 文件路径（所有重要文件的路径）
- 📝 日志位置（各姐妹的日志文件）
- ⚙️ 自动化脚本（同步脚本、自动化系统）
- 📊 姐妹状态（Gateway 状态、发帖能力等）
- 🚀 开发进度（记忆系统开发进度）
- 📄 开发文档（所有开发相关文档）

**关键信息汇总文件：** `/home/micker/clawd/memory/姐妹们关键信息.md`
